Demisauce is a set of Open Source services: "Its a feature not a Site"

Services that are easily embeddable into your application, not a site you need to send your user's to get those features. It is is available as a service so it is not another framework to build on top of. Services can be either server side (with caching) or browser service's (javascript).


Comment System

A comment system that can be included inside an application or content to provide javascript enabled commenting. See blog article explaining a bit more.

Help, Ideas, Feedback

If you have wanted to manage content for display in your existing application but not wanted the Page Templating, and frameworks that they come with, this is the content management, without page templating. More about Feedback...

Polls

Although other consumer oriented servie make it easy to embed polls with javascript into your site, this may not suit apps that do not want the polls public.

Activity Tracking

Easily embed single javascript file, and track your users behavior and provide embeddable visualizations for your applications.

Technology

The demisauce server is separate from the demisauce client api libraries. Currently there are python and PHP implementations of the library with a partial .net implementation.
