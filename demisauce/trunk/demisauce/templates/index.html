Demisauce is a set of Open Source services: "Its a feature not a Site"

Services that are easily embeddable into your application, not a site you need to send your user's to get those features. Services can be either server side (with caching) or browser service's (javascript).

About

This project started out of seeing the custom software applications (Java/.NET mostly) where the core developers worked. So much focus was placed on core needs, that periphery things like email templates, content, integrated help, feedback tools etc took a back seat.

Services for your App via SOA

Web based administration tools to manage content. Access the content from your application through easy client libraries. Or, browser to server services...
