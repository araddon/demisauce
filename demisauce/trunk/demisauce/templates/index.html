Demisauce is an Open Source services tool for delivering Content Management, Knowledge base, Email Templates, and via Web services for use within existing applications.

Admin tools to administer content which is offered over web services to be displayed within your applications.

Comment System

A comment system that can be included inside an application or content to provide javascript enabled commenting. See blog article explaining a bit more.

<%call expr="self.box2()">

Content Management

If you have wanted to manage content for display in your existing application but not wanted the Page Templating, and frameworks that they come with, this is the content management, without page templating.

<%call expr="self.box2()">

Email Template Management

Email templates seem to get embedded in code, text files, databases, and numerous other places all over many different software applications.
This service provides admin tools to manage the templates, but getting them is a simple web service call (cached) using a library.

About

This project started out of seeing the custom software applications (Java/.NET mostly) where the core developers worked. So much focus was placed on core needs, that periphery things like email templates, content, integrated help, feedback tools etc took a back seat.

How does it work?

There is a python, and also a DotNet (C#) client's to talk to the web services. There is also a DotNet (C#) demo project that utilizes the cms capabilities.
